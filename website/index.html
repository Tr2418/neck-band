System

How CervicalSentinel Works

Five steps from sensor to alert — all happening in real-time.

1

Sense

MyoWare EMG electrodes placed on the upper trapezius muscle capture electrical signals at 500 Hz. An MPU6050 IMU simultaneously measures neck tilt and shoulder angles.

2

Process

The ESP32 runs a windowed RMS filter on raw EMG data (50-sample window) and a complementary filter (α=0.96) on IMU data to remove noise and drift.

3

Classify

An on-device AI decision tree classifies muscle strain into 4 levels (Normal → High) and posture into 4 classes (Good → Combined). Alert level 0–2 is assigned.

4

Transmit

A compact JSON packet is sent once per second via BLE GATT notification to the mobile app: {"e":350,"n":12.5,"s":8.2,"sc":0,"pc":0,"al":0,"b":85}

5

Alert

The mobile app displays live data, persists alerts and session history, and vibrates/notifies when posture correction is needed.

Hardware

What's Inside the Band

Three components work together to give you complete monitoring.

⚡

MyoWare 2.0 EMG

Primary Sensor

Measures electrical signals from upper trapezius, SCM, and levator scapulae muscles. Detects muscle fatigue before visible posture change.

  • 500 Hz sampling rate
  • Windowed RMS processing
  • Baseline drift compensation
🖥️

ESP32

Microcontroller + AI

Runs the on-device AI classifier, BLE server, hardware timers, and battery monitoring. The brain of the wearable.

  • Dual-core 240 MHz
  • BLE 5.0 GATT server
  • 12-bit ADC, hardware timer
📐

MPU6050 IMU

Secondary Sensor

6-axis accelerometer + gyroscope measures neck tilt and shoulder rounding angles via complementary filter fusion.

  • 50 Hz update rate
  • Complementary filter (α=0.96)
  • I²C interface, 3.3 V

Setup Guide

Get Started — Step by Step

Complete instructions for Windows 11 users running VS Code. No prior experience needed.

1

Install Git for Windows

Go to git-scm.com/download/win and download the installer. Run it and click Next through all steps (defaults are fine).

2

Clone the Repository

Open VS Code → press Ctrl + ` to open the terminal → paste this command and press Enter:

git clone https://github.com/Tr2418/neck-band.git
3

Open the Folder in VS Code

In VS Code go to File → Open Folder and select the neck-band folder that was just created.

4

Open the Website

In the VS Code file explorer on the left, click website → then double-click index.html. A preview will open, or you can right-click the file and choose "Open with Live Server" if you have the Live Server extension installed.

To install Live Server: press Ctrl+Shift+X → search Live Server → click Install.

5

View in Browser

Right-click website/index.html in VS Code → Open with Live Server. Your default browser will open with the website running at http://127.0.0.1:5500/website/index.html.

1

Install Node.js

Go to nodejs.org → click LTS (recommended) → download and run the installer. Accept all defaults and click Next.

Verify it installed: in VS Code terminal type node -v → should print a version like v20.x.x.

2

Install Expo CLI

In the VS Code terminal, run:

npm install -g expo-cli
3

Navigate to the App Folder

cd neck-band/CervicalSentinel
4

Install Dependencies

npm install

This downloads all the libraries the app needs. It may take 1–2 minutes.

5

Start the App

npx expo start

A QR code will appear in the terminal.

6

Open on Your Phone

Install Expo Go from the App Store or Play Store on your phone. Open Expo Go → tap Scan QR Code → scan the QR code in your terminal. The app loads on your phone!

1

Install Arduino IDE 2

Go to arduino.cc/en/software → download Arduino IDE 2.x for Windows → run the installer.

2

Add ESP32 Board Package

Open Arduino IDE → File → Preferences → paste this URL in the Additional Board Manager URLs box:

https://raw.githubusercontent.com/espressif/arduino-esp32/gh-pages/package_esp32_index.json

Then: Tools → Board → Boards Manager → search esp32 → Install esp32 by Espressif.

3

Open the Firmware Sketch

In Arduino IDE: File → Open → navigate to neck-band/firmware/CervicalSentinel.ino → click Open.

4

Select Board & Port

Tools → Board → ESP32 Arduino → ESP32 Dev Module
Tools → Port → select the port that appeared when you plugged in the ESP32 (e.g. COM3).

5

Upload

Click the → Upload button (or press Ctrl+U). Wait for "Done uploading". Open Tools → Serial Monitor at 115200 baud to see live data.

1

Go to Your Repository on GitHub

Open github.com/Tr2418/neck-band in your browser.

2

Open Repository Settings

Click the Settings tab at the top of the repo page.

3

Enable GitHub Pages

Scroll down to the Pages section in the left sidebar → click it.

Under Source, select Deploy from a branch → Branch: main → Folder: / (root) or set it to /website if that option appears → click Save.

4

Wait ~1 Minute

GitHub will build and deploy your site automatically. Refresh the Settings → Pages page and you'll see a green banner with your live URL:

https://tr2418.github.io/neck-band/website/
5

Share Your Site 🎉

Copy that URL and share it with anyone — they can view your CervicalSentinel website from any device without installing anything.
